# Stay Focused

Background process for staing focused. Killing distracting applications during working day.

### Config file

    # ~/.config/.stay_focused.json
    {
        "time": {
            "start": "09:00",
            "start": "18:00"
        },
        "applications": [
            "skype",
            "viber",
            ...,
            "telegram"
        ],
    }
